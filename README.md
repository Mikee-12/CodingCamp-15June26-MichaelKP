# CodingCamp-15June26-MichaelKP

## Features

### Greeting (can be renamed by clicking the name displayed in the header)
### Focus Timer (Pomodoro)
### To-Do List
### Quick Links

## Additional Features

### Theme Customization
- Light Mode
- Dark Mode

### Enhanced Task Management
- Duplicate task prevention.
- Task sorting.
- Priority task support.
- Task editing.
- Task deletion.

### Gamification
- Earn points for every completed task.

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the application in your browser.
2. Click the displayed name in the header to customize it.
3. Add tasks to your To-Do List.
4. Set task priorities and organize them as needed.
5. Start the Pomodoro timer and customize the focus duration.
6. Complete tasks to earn points.
7. Use Quick Links for easy access to frequently visited websites.

## Project Goals
This project was created to combine task management, focus tracking, and productivity enhancement into a simple, user-friendly web application.
